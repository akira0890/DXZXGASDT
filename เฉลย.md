## ข้อ 1 — ตอบ: Sc

- อัตราส่วนปฏิกิริยา nitration ของ toluene → TNT คือ 1 : 3 (toluene + 3HNO₃ → TNT + 3H₂O)
  ⇒ ไอออนของสารทำปฏิกิริยากับ [Ag(CN)₂]⁻ แบบ 1 : 3 ⇒ ไอออนมีประจุ **+3**
- ใช้กฎแก๊สอุดมคติหามวลโมเลกุลของไอออน:
  - n = PV/RT = (4.12×10⁻¹⁶ bar × 0.9869 atm/bar × 1 L)/(0.08206 × 298.15) = 1.66×10⁻¹⁷ mol
  - M = 2.54×10⁻¹⁵ g ÷ 1.66×10⁻¹⁷ mol ≈ **152.8 g/mol**
- รูปทรง octahedral + 3 ธาตุ + ประจุ +3 + M≈153 ⇒ **[Sc(H₂O)₆]³⁺** (= 153.05 g/mol) → ธาตุ 3 ชนิดคือ **Sc, H, O**
- ต่อเซลล์กับ SHE: เข็มชี้ไปทางบีกเกอร์ SHE ⇒ SHE เป็นขั้วบวก (แคโทด) ⇒ ธาตุนั้นเป็นแอโนด (E° < 0)
  - Sc: E° = −2.08 V (< 0) ✓  | O₂/H₂O: +1.23 V (> 0) ✗ | H: = 0
  - ⇒ ธาตุคำตอบคือ **Sc**

## ข้อ 2 — ตอบ: aa (กรดอะมิโน / amino acid)

- อุกกาบาต Murchison, ALH84001, Orgueil, Tagish Lake → ทฤษฎี **Panspermia** (กำเนิดชีวิต/สารตั้งต้นชีวิตมาจากอวกาศ)
- สิ่งมีชีวิตปริศนา (ผนังเซลล์ β-1,4-glucan/เซลลูโลส ไม่มีไคติน, zoospore 2 แฟลเจลลัมแบบ heterokont, คล้ายรา) = **Oomycete (ราน้ำ)**
- องค์ประกอบร่วม = **กรดอะมิโน (aa)** — อุกกาบาตมีกรดอะมิโนนอกโลก, Oomycete มีกรดอะมิโน (เด่นคือ hydroxyproline ที่ผนังเซลล์)
- จุดเชื่อมเชิงลึก: กรดอะมิโนในอุกกาบาตมี **L-enantiomeric excess** ตรงกับสิ่งมีชีวิตที่ใช้กรดอะมิโนรูป **L** (homochirality)

## ข้อ 3.1 — ตอบ: N_A·E_i = b × 10⁻³ J/mol

- P_i = (1/Z)·e^(−E_iβ_i), Z = 1 ⇒ ln P_i = −E_i/(k_BT) = −c/a
- ⇒ E_i = (c/a)·k_BT ⇒ N_A E_i = (c/a)·(N_A k_B)T = (c/a)·R·T
- กฎแก๊สอุดมคติ: RT = PV/n = (b)(a×10⁻³ m³)/(c) ⇒ **N_A E_i = (c/a)·(ab×10⁻³/c) = b×10⁻³ J/mol** (a, c ตัดกันหมด)

## ข้อ 3.2 — ตอบ: Y (อิตเทรียม)

- ปริมาตรเตา = πr²h = π(1/2π)²(16π) = **4 m³** ⇒ V = N = 4 (อะตอม/โมเลกุล), n = 8V = **32 mol**
- แก๊ส polyatomic 4 อะตอม: C_V = (3N−3)R = 9R ⇒ γ = C_P/C_V = 10/9
- v² = γP/ρ ⇒ ρ = γP/v² = (10/9)(120000)/291.82² = 1.5657 kg/m³
- ρ = nM/V ⇒ M = ρV/n = 1.5657×4/32 = 0.19571 kg/mol = **195.7 g/mol**
- ธาตุ EA สูงสุด = **Cl** ; สาร 2 ธาตุ 4 อะตอม = XCl₃ ⇒ X = 195.7 − 3(35.45) ≈ 89.4 → **Y** (88.9) ⇒ **YCl₃**

## ข้อ 4 — ตอบ: arccos(z) = 2.8171 − 0.3226 i (เรเดียน)

- จากรูป จุดกลาง = **จุดศูนย์กลางวงกลม O** (เส้นตั้งฉากจาก O ลงคอร์ด = เส้นแบ่งครึ่งตั้งฉาก)
- θ₁ = มุมในวงกลม ∠LTB รองรับส่วนโค้ง LB ⇒ มุมที่จุดศูนย์กลาง ∠LOB = 2θ₁
- ΔOLB หน้าจั่ว (OL = OB = R) ⇒ θ₂ = ∠OLB = (180° − 2θ₁)/2 = **90° − θ₁ = 88°**
- 2θ₂ = 176° = 180° − 2θ₁ ⇒ e^(2iθ₂) = −cos2θ₁ + i·sin2θ₁
- z = i·sinθ₁ + e^(2iθ₂) = −cos2θ₁ + i(sin2θ₁ + sinθ₁) = **−0.99756 + 0.10466 i**
- **\|z\| = 1.00304 > 1** ⇒ ใช้ arccos จริงไม่ได้ ต้องใช้ arccos เชิงซ้อน:
  - arccos(z) = −i·Ln(z + i√(1−z²)) = **2.81709 − 0.32262 i** (เรเดียน)
  - ส่วนจริง ≈ 161.41° ; ตรวจกลับ cos(2.81709 − 0.32262 i) = z ✓
- ค่าทั่วไป: arccos(z) = ±(2.81709 − 0.32262 i) + 2πk
