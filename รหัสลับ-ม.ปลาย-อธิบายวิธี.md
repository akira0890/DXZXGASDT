# ข้อ 1 — เคมีโคออร์ดิเนชัน + เคมีไฟฟ้า → ธาตุ **Sc**

## 1) ถอดเงื่อนไขทีละข้อ
| เงื่อนไข | สื่อถึง |
|---------|--------|
| ประกอบด้วยธาตุ 3 ชนิด, ลิแกนด์ monodentate, ทรง Octahedral | สารเชิงซ้อน **[M(L)₆]** ลิแกนด์ตัวเดียวจับ 1 ตำแหน่ง 6 ตัว |
| จับ [Ag(CN)₂]⁻ อัตราส่วนเท่าปฏิกิริยา nitration ของ toluene → TNT | toluene + **3** HNO₃ → TNT + 3H₂O ⟹ อัตราส่วน **1 : 3** ⟹ ไอออนบวกมีประจุ **3+** |
| เกิดสารละลาย **ไม่มีสี** | คอมเพล็กซ์ **d⁰** (ไม่มี d–d transition) |
| ลิแกนด์ + โลหะ = 3 ธาตุ | ถ้าลิแกนด์เป็น **H₂O** → ธาตุ M, H, O ครบ 3 |

## 2) หามวลโมเลกุลจากกฎแก๊สอุดมคติ (ยืนยันว่าโลหะคือ Sc)
ข้อมูล: T = 298.15 K, V = 1 dm³ = 1 L, P = 4.12×10⁻¹⁶ bar, มวลไอออน = 2.54×10⁻¹⁵ g, R = 0.08206 atm·L·mol⁻¹·K⁻¹

$$n=\frac{PV}{RT}=\frac{(4.12\times10^{-16})(1)}{(0.08206)(298.15)}\approx 1.66\times10^{-17}\ \text{mol}$$

$$M=\frac{\text{มวล}}{n}=\frac{2.54\times10^{-15}}{1.66\times10^{-17}}\approx \mathbf{153\ g/mol}$$

เทียบ [M(H₂O)₆]³⁺ = M + 6(18.02):

| ไอออน | มวล (g/mol) |
|------|------------|
| **[Sc(H₂O)₆]³⁺** | **153.05** ✅ |
| [Ti(H₂O)₆]³⁺ | 155.96 |
| [Al(H₂O)₆]³⁺ | 135.07 |

→ โลหะคือ **Sc** (สแกนเดียม). Sc³⁺ = [Ar]3d⁰ → **ไม่มีสี** ✓ ตรงทุกเงื่อนไข
สารประกอบเต็มคือ **[Sc(H₂O)₆][Ag(CN)₂]₃** ; 3 ธาตุที่แยกได้ = **Sc, H, O**

## 3) เซลล์กัลวานิกกับ SHE → เลือกธาตุคำตอบ
นำ Sc, H, O ทำขั้วไฟฟ้าเทียบ **SHE (E° = 0)** ในกรด:

| ขั้วจากธาตุ | ครึ่งปฏิกิริยา | E° (V) | เทียบ SHE |
|------------|--------------|-------|-----------|
| **Sc** | Sc³⁺ + 3e⁻ ⇌ Sc | **−2.08** | ต่ำกว่า SHE → Sc เป็น **แอโนด**, SHE เป็นแคโทด(+) |
| H | 2H⁺ + 2e⁻ ⇌ H₂ | 0.00 | คือ SHE เอง |
| O | O₂ + 4H⁺ + 4e⁻ ⇌ 2H₂O | +1.23 | สูงกว่า SHE → SHE เป็นแอโนด |

"เข็มโวลต์มิเตอร์ชี้ไปทางบีกเกอร์ SHE" = SHE เป็นขั้วบวก (แคโทด) ⟹ ธาตุต้องมี **E° < 0**
มีเพียง **Sc** → **ธาตุคำตอบ = Sc**  → ตัวอักษร **`Sc`** (สองตัวแรกของ SCOOBY)

# ข้อ 2 — ดาราศาสตร์ชีววิทยา → **oo**

## 1) ทฤษฎีจากอุกกาบาต 4 ลูก
Murchison (1969), ALH84001 (Mars), Orgueil (1864), Tagish Lake (2000) — ล้วนเป็น **คาร์บอนาเชียสคอนไดรต์/อุกกาบาตดาวอังคาร** ที่พบ "โมเลกุลอินทรีย์" (กรดอะมิโน น้ำตาล เบสไนโตรเจน) และเคยถูกอ้างว่าพบ "ร่องรอยสิ่งมีชีวิต"
→ ทฤษฎี **Panspermia** (ชีวิต/สารตั้งต้นของชีวิตเดินทางมาจากอวกาศ)

## 2) สิ่งมีชีวิตปริศนา
- วงจรชีวิต**คล้ายรา** แต่ **ไม่มี N‑acetylglucosamine (ไคติน)**
- ผนังเซลล์เป็น **β‑1,4‑glucan (เซลลูโลส)**
- **zoospore มี 2 flagellum แบบ heterokont** (เส้นหนึ่งมีขน tinsel อีกเส้น whiplash)

ลักษณะเหล่านี้ = **Oomycete (ราน้ำ / Oomycota)** — "ราเทียม" ในกลุ่ม Stramenopila ญาติกับสาหร่ายสีน้ำตาล/ไดอะตอม

## 3) จุดร่วม (2 ตัวอักษร)
- **Pan‑sperm‑ia** = กรีก *pan* (ทั้งหมด) + *sperma* (**เมล็ด/สเปิร์มแห่งชีวิต**)
- **Oo‑mycete** = กรีก *ōon* (**ไข่**) + *mykes* (รา)

ทั้งสองสื่อถึง "เมล็ด/ไข่ของชีวิต" → ตัวอักษร **`oo`**
(องค์ประกอบสะพานเชื่อม = **กรดอะมิโน/ไกลซีน** ที่พบทั้งในอุกกาบาตทั้ง 4 และในสิ่งมีชีวิต)

# ข้อ 3 — กลศาสตร์สถิติ/อุณหพลศาสตร์ (ธีม Scooby‑Doo) → **by**

## 3.1 หา N_A⁻¹Eᵢ → ได้ "b"
Boltzmann: $P_i=\frac{1}{Z}e^{-E_i\beta_i}$ , $\beta_i=\frac{1}{k_BT}$ , Z = 1

$$\ln P_i=-\frac{E_i}{k_BT}=-\frac{c}{a}\quad\Rightarrow\quad E_i=\frac{c}{a}k_BT$$

กฎแก๊สอุดมคติ (P=b, V=a, n=c): $\;T=\dfrac{PV}{nR}=\dfrac{ba}{cR}$

แทนค่า (a, c ตัดกันหมด) และใช้ $k_B/R = 1/N_A$:

$$E_i=\frac{c}{a}\,k_B\cdot\frac{ba}{cR}=\frac{k_B\,b}{R}=\boxed{\frac{b}{N_A}}\qquad\Rightarrow\qquad \boxed{N_A^{-1}E_i=\frac{b}{N_A^{2}}}$$

ผลลัพธ์เหลือเพียงตัวแปร **b** → ตัวอักษร **`b`**

## 3.2 หา (v_sound/v_rms)² → ได้ "y" (γ)
- ความเร็ว rms: $v_{rms}=\sqrt{\dfrac{3RT}{M}}$
- ความเร็วเสียง: $v_{sound}=\sqrt{\dfrac{\gamma RT}{M}}$

$$\left(\frac{v_{sound}}{v_{rms}}\right)^2=\frac{\gamma RT/M}{3RT/M}=\frac{\gamma}{3}=\frac{5/3}{3}=\boxed{\frac{5}{9}}\quad(\text{แก๊สอะตอมเดี่ยว }\gamma=\tfrac{5}{3})$$

หัวใจของข้อนี้คือ **ค่าคงตัวเชิงอุณหพลศาสตร์ γ (แกมมา)** ซึ่งสัญลักษณ์ **γ ≈ "y"** → ตัวอักษร **`y`**
*(หมายเหตุ: ประโยค "คูณงาน isothermal และ isochoric ส่วน 2R" เขียนกำกวม — งาน isochoric = 0 เป็น "กับดัก"; ถ้าบังคับให้ได้ค่าคงที่เดียวจริง ตัวที่เป็นไปได้สุดคือ 5/9 × C_p/2R = 25/36 แต่ payload ของรหัสคือ γ → "y")*

**รวม 3.1 + 3.2 = `b` + `y` = `by`**

# ข้อ 4 — จำนวนเชิงซ้อน + มุมในวงกลม → **Doo**

โจทย์: $\Delta\arccos\!\big(i\sin\theta_1+e^{2i\theta_2}\big)$ , θ₁ = 2°

## ตัวอักษรของรหัส
- **Δ = อักษรกรีก "Delta" = "D"** (และรูปในโจทย์คือ "สามเหลี่ยม" ซึ่งใช้สัญลักษณ์ Δ พอดี)
- ส่วน arccos(...) → **"oo"**
- → **`Doo`** (สามตัวสุดท้ายของ SCOOBY‑**DOO**)

## เชิงคณิตศาสตร์ (สำหรับคนอยากเห็นตัวเลข)
แยกด้วยออยเลอร์: $e^{2i\theta_2}=\cos2\theta_2+i\sin2\theta_2$

$$z=i\sin\theta_1+e^{2i\theta_2}=\cos2\theta_2+i(\sin\theta_1+\sin2\theta_2)$$

ถ้าเรขาคณิต (มุมในวงกลม + มุมฉากในรูป) ให้ความสัมพันธ์ **2θ₂ = 180° + θ₁** จะได้
$e^{2i\theta_2}=-e^{i\theta_1}=-\cos\theta_1-i\sin\theta_1$ ดังนั้น

$$z=i\sin\theta_1-\cos\theta_1-i\sin\theta_1=-\cos\theta_1=-\cos2^\circ$$

$$\arccos(-\cos2^\circ)=180^\circ-2^\circ=\boxed{178^\circ}$$

(เป็นค่าที่ "สะอาด" ที่สุดในตระกูลนี้ และคืนค่า θ₁ กลับมาพอดี: 178 = 180 − 2)
แต่ในบริบทรหัสลับ ตัวที่ต้องนำไปต่อคือ **`Doo`**
