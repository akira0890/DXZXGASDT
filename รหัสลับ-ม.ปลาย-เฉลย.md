## 🔑 รหัสลับสุดท้าย = **SCOOBY‑DOO** 🐕

ปริศนานี้ทั้งชุดมีธีม **Scooby‑Doo** (ข้อ 3 พูดถึง "แก๊ง Scooby‑Doo" และ "รถ Mystery Machine" ตรง ๆ)
นำคำตอบข้อ 1–4 มาต่อกันได้ชื่อ **SCOOBYDOO**

| ข้อ | คำตอบ | ตัวอักษร |
|-----|-------|----------|
| 1 | ธาตุ **Sc** (สแกนเดียม) | `Sc` |
| 2 | **oo** (panspermia ↔ **oo**mycete) | `oo` |
| 3 | **b** + **y** (γ) | `by` |
| 4 | **Δ**(Delta=D) + arccos → oo | `Doo` |

```
S c  o o  b y  D o o   →   SCOOBYDOO
└P1┘ └P2┘ └P3┘ └─P4─┘
```

---

## คำตอบรายข้อ

### ข้อ 1 → **Sc** (สแกนเดียม)
- สารประกอบคือ **[Sc(H₂O)₆]³⁺** (ไอออนเฮกซะอะควาสแกนเดียม(III))
  - มวลโมเลกุลจากกฎแก๊ส ≈ **153 g/mol** ตรงกับ [Sc(H₂O)₆]³⁺ = 153.05
  - Sc³⁺ เป็น **d⁰ → ไม่มีสี**, ทรงแปดหน้า (octahedral), ลิแกนด์ H₂O แบบ monodentate
  - ประจุ 3+ จับ [Ag(CN)₂]⁻ อัตราส่วน **1 : 3** (= toluene : HNO₃ ในการสร้าง TNT)
  - 3 ธาตุ = **Sc, H, O**
- เซลล์กัลวานิกกับ SHE: ในสามธาตุ มีเพียง **Sc (E° = −2.08 V < 0)** ที่เป็นขั้วแอโนด ทำให้ SHE เป็นขั้วบวก → เข็มโวลต์มิเตอร์ชี้ไปทาง SHE
- **ธาตุคำตอบ = Sc** ✅ *(มั่นใจสูงมาก)*

### ข้อ 2 → **oo**
- อุกกาบาต 4 ลูก (Murchison, ALH84001, Orgueil, Tagish Lake) → ทฤษฎี **Panspermia** (สิ่งมีชีวิต/สารตั้งต้นมาจากอวกาศ)
- สิ่งมีชีวิตปริศนา (คล้ายรา, ไม่มีไคติน, ผนัง β‑1,4‑glucan/เซลลูโลส, zoospore 2 แฟลเจลลัม heterokont) = **Oomycete (ราน้ำ)**
- จุดร่วม: pan‑**sperm**‑ia (เมล็ด/สเปิร์มแห่งชีวิต) ↔ **Oo**‑mycete ("oo" = ไข่) → **oo** ✅

### ข้อ 3 → **by**
- **3.1:** จาก Boltzmann (Z=1): ln Pᵢ = −Eᵢ/(k_BT) = −c/a และกฎแก๊ส T = ba/(cR)
  → **Eᵢ = b/N_A** ดังนั้น **N_A⁻¹Eᵢ = b/N_A²** (a, c ตัดกันหมด เหลือ **b**) → ตัวอักษร **b**
- **3.2:** (v_sound/v_rms)² = **γ/3 = 5/9** (แก๊สอะตอมเดี่ยว γ = 5/3) → ค่าที่ได้คือ **γ (แกมมา ≈ "y")** → ตัวอักษร **y**
- รวม → **by** ✅

### ข้อ 4 → **Doo**
- **Δarccos( i·sinθ₁ + e^{2iθ₂} )**, θ₁ = 2°
- **Δ = อักษรกรีก Delta = "D"** (และรูปในโจทย์คือสามเหลี่ยม = สัญลักษณ์ Δ)
- ส่วน arccos(...) → **"oo"** → รวมเป็น **Doo** ✅
- *(หมายเหตุ: ถ้าให้คิดเชิงเลขจริง โดยเรขาคณิตให้ 2θ₂ = 180° + θ₁ จะได้ z = −cosθ₁ และ arccos(−cos2°) = **178°** พอดี — แต่ payload ของรหัสคือ "Doo")*

## สรุประดับความมั่นใจ

| ส่วน | สถานะ |
|------|-------|
| รหัสรวม = **SCOOBYDOO** | ✅ **มั่นใจสูงมาก** (ธีม Scooby‑Doo ระบุชัดในข้อ 3) |
| ข้อ 1 = Sc (เคมีครบสูตร) | ✅ **มั่นใจสูงมาก** (มวล 153 = [Sc(H₂O)₆]³⁺) |
| ข้อ 2 = oo | ✅ ยืนยันด้วยธีม + นิรุกติศาสตร์ oomycete |
| ข้อ 3 = by (b, γ→y) | ✅ ฟิสิกส์ b/N_A และ γ/3 = 5/9 แน่นอน |
| ข้อ 4 = Doo (Δ→D) | 🟡 Δ=D ชัด; "oo" อนุมานจากธีม (ฉบับร่าง crude) |

> ⚠️ ข้อ 3.2 และข้อ 4 ในต้นฉบับเขียนกำกวม (ระบุว่าเป็นร่าง "crude") — แต่เพราะรหัสรวมต้องเป็น **SCOOBYDOO** ตัวอักษรของแต่ละข้อจึงถูกล็อกไว้
